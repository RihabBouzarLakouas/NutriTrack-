# NutriTrack  

## Introduction
Le projet **NutriTrack** a pour objectif de promouvoir une alimentation saine et équilibrée à travers une application open source.

Cette application aide les utilisateurs à suivre leurs habitudes alimentaires, à calculer leurs besoins nutritionnels quotidiens et à adopter un mode de vie plus sain.  
Elle offre une approche simple, interactive et accessible à tous.

## Description du projet
**NutriTrack** est conçu comme un outil de suivi nutritionnel intelligent.  
Il permet d’enregistrer les repas consommés, d’analyser les apports caloriques, et de générer des recommandations personnalisées selon le profil de chaque utilisateur.  
Le projet vise à sensibiliser sur l’importance d’une bonne nutrition et à faciliter la gestion quotidienne de l’alimentation.
